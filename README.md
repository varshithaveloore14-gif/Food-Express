# 🍔 Food Express - Food Delivery UI

A modern and responsive **Food Delivery Website UI** built using **HTML5** and **CSS3**.
This project showcases a clean layout design for a food ordering platform.

---

## 🚀 Live Demo

👉 https://varshithaveloore14-gif.github.io/Food-Express/

---

## ✨ Features

* 📱 Fully responsive design (mobile, tablet, desktop)
* 🍕 Attractive food layout and sections
* 🎨 Clean and modern UI design
* 🎯 Smooth hover effects and styling
* 📦 Structured layout for real-world food apps

---

## 🛠️ Technologies Used

* **HTML5** — Structure of the webpage
* **CSS3** — Styling, layout (Flexbox/Grid), and effects

---

## 📂 Project Structure

```bash
Food-Express/
│── index.html
│── style.css
```

---

## ⚙️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/varshithaveloore14-gif/Food-Express.git
```

2. Navigate to the folder:

```bash
cd Food-Express
```

3. Open in browser:

```bash
open index.html
```
